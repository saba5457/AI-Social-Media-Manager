# 📱 AI Social Media Manager

AI-powered content generator for 5 platforms — human-style, no AI feel.

## ✨ Features
- Instagram captions
- LinkedIn professional posts
- Twitter/X tweets (under 280 chars)
- TikTok captions & script ideas
- Facebook community posts
- Full Pack — all 5 platforms at once
- 5 tone options: Casual, Professional, Witty, Emotional, Minimal
- One-click Copy button

## 🛠️ Tech Stack
- Python + Flask
- LangChain + Groq API (llama-3.1-8b-instant)
- HTML + CSS + JavaScript

## 🚀 How to Run
1. `pip install -r requirements.txt`
2. Add `GROQ_API_KEY=your_key` in `.env` file
3. `python app.py`
4. Open: `http://127.0.0.1:5003`
